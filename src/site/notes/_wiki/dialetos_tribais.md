**Dialetos Tribais**: Sem descrição fonológica possível como categoria. Cada raça híbrida desenvolveu sistemas próprios, frequentemente incompatíveis entre si. O denominador comum é a funcionalidade: dialetos tribais existem para comunicação imediata dentro de comunidades pequenas, sem ambição literária ou ritual.
