# Sistema de Magia

Toda magia exige pagamento. O corpo do conjurador é a moeda.

---

## Sistemas de Acesso

As vias pelas quais mortais canalizam poder arcano:

- [[_wiki/teru\|Teru]] — Estudo técnico, compreensão arcana pura
- [[_wiki/pactum_diaboli\|Pactum Diaboli]] — Contrato com diabos
- [[_wiki/vinculum_daemonis\|Vinculum Daemonis]] — Vínculo com demônios
- [[_wiki/devotio_sancta\|Devotio Sancta]] — Devoção divina
- [[_wiki/symbiosis_naturae\|Symbiosis Naturae]] — Simbiose com a natureza
- [[_wiki/meditatio_aetheris\|Meditatio Aetheris]] — Meditação etérea

## Escolas de Magia

Trinta escolas, divididas pelas cinco cores do espectro arcano. Cada escola possui dez feitiços catalogados.
