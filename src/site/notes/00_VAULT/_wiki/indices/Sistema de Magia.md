# Sistema de Magia

Toda magia exige pagamento. O corpo do conjurador é a moeda.

---

As vias pelas quais mortais canalizam poder arcano:

- [[00_VAULT/_wiki/magia/teru\|Teru]] — Estudo técnico, compreensão arcana pura
- [[00_VAULT/_wiki/magia/pactum_diaboli\|Pactum Diaboli]] — Contrato com diabos
- [[00_VAULT/_wiki/magia/vinculum_daemonis\|Vinculum Daemonis]] — Vínculo com demônios
- [[00_VAULT/_wiki/magia/devotio_sancta\|Devotio Sancta]] — Devoção divina
- [[00_VAULT/_wiki/magia/symbiosis_naturae\|Symbiosis Naturae]] — Simbiose com a natureza
- [[00_VAULT/_wiki/magia/meditatio_aetheris\|Meditatio Aetheris]] — Meditação etérea

Trinta escolas, divididas pelas cinco cores do espectro arcano. Cada escola possui dez feitiços catalogados.
